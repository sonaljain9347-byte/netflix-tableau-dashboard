# netflix-tableau-dashboard
## Project Description
This project is an interactive Netflix Analytics Dashboard created using Tableau. It analyzes more than 8,800 Netflix Movies and TV Shows to provide insights into content distribution, genres, ratings, countries, release years, and overall trends. The dashboard helps users explore Netflix data through interactive filters and visualizations.

## Tools Used
- Tableau Public
- GitHub
- Microsoft Excel / CSV Dataset

## Dataset
- Netflix Movies and TV Shows Dataset
- Total Records: 8,800+

## Key Performance Indicators (KPIs)
- Total Titles
- Movies vs TV Shows
- Top Genres
- Top Ratings
- Content by Country
- Year-wise Content Growth

## Dashboard Features
- Content distribution by type
- Top genres analysis
- Country-wise content analysis
- Rating distribution
- Release year trend
- Interactive filters and charts
